🌀 Naruto Chakra Control System

«"Hands for power…"»

A real-time hand tracking web project that lets you perform Naruto-style jutsus using your camera.

This system uses:

- ✋ Hands → to generate chakra

🎥 Demo

Use your hands to unlock abilities:

- 🔵 Rasengan (Right Palm)
- 🟣 Dark Chakra (Left Palm)
  

⚙️ Features

- ✋ Real-time hand tracking
- 🌀 Chakra-based visual effects
- 🎯 Minimal gesture system (clean & responsive)
- 🌐 Runs directly in browser


🎮 Controls

Action| Power
✋ Right Open Palm| Rasengan
✋ Left Open Palm| Dark Chakra
✋ Both Hands| Power Boost Mode

🧠 Tech Stack

- JavaScript
- HTML / CSS
- MediaPipe Hands
- MediaPipe Face Mesh


🌀 Power System

🔵 Rasengan

螺旋丸 (Rasengan)
A spinning sphere of chakra formed in the right hand.


🟣 Dark Chakra

闇遁 (Dark Style)
A dense and unstable energy formed in the left hand.


⚡ Dual Power Mode

When both hands are active, chakra becomes unstable and stronger.



🚀 How to Run

💻 PC (Recommended)

1. Open project in VS Code
2. Run "index.html" using Live Server


📱 Mobile

1. Run using local server
2. Allow camera access
3. Use in Chrome browser


⚠️ Requirements

- Camera access enabled
- Good lighting on hands and face
- Stable background
